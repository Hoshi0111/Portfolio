## HEAL
５人でのチーム開発<br>
・概要<br>
HEAL(Healthcare Empowerment through AI Linguistic)<br>
留学生などが自国以外で診療を受ける際、医師の質問や治療の説明をよく理解できない不安を解消するための診療補助システム<br>
問診、診察チャットをそれぞれの母国語に変換<br>
医療用語をAIが判別し、医療用語データベースから解説を提示<br>
